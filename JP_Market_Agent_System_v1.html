사용 가이드

이 시스템은 무엇인가

이 HTML은 일본 시장 인텔리전스를 반복적으로 수집·분석·축적하는 AI Agent 시스템입니다.
5개의 전문 Agent 프롬프트를 Claude Max에 복사-붙여넣기하면 최신 시장 분석이 실행됩니다.
분석 결과는 이 대시보드의 Intelligence 탭에 축적하여 시간 경과에 따른 변화를 추적합니다.

핵심 원칙: 경쟁사의 세세한 활동이 아닌, "우리 브랜드가 어떻게 성장할 것인가"에 집중합니다.

1

Agent Prompts 탭에서 프롬프트 복사

실행할 Agent의 "Copy" 버튼을 클릭하여 프롬프트를 복사합니다.
주간 루틴: 월(Agent 1) → 수(Agent 3) → 금(Agent 5)

2

Claude Max에 붙여넣기 → 실행

Claude Max (claude.ai) 새 대화에 프롬프트를 붙여넣고 실행합니다.
Claude가 웹서치를 활용하여 최신 시장 데이터를 수집·분석합니다.

3

핵심 인사이트를 Intelligence 탭에 기록

분석 결과 중 "우리 브랜드에 의미있는 것"만 Intelligence 탭에 추가합니다.
"+ 인사이트 추가" 버튼으로 날짜·카테고리·내용·시사점을 기록합니다.

4

Agent 5로 주간/월간 전략 브리핑 생성

Agent 1~4의 분석 결과를 Agent 5 프롬프트에 붙여넣으면
종합 전략 브리핑이 생성됩니다. 이것이 의사결정 지원 자료입니다.

5

프롬프트는 자유롭게 수정 가능

시장 상황이 변하거나, 새로운 분석이 필요하면 프롬프트를 직접 수정하세요.
예: 새로운 경쟁사 등장 시 Agent 3에 해당 브랜드 추가, 신규 채널 검토 시 Agent 3 확장.
